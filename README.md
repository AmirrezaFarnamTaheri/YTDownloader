# YTDownloader

YTDownloader is a simple and efficient command-line tool for downloading videos from YouTube. This project is built with Python and utilizes the powerful `yt-dlp` library to provide a seamless video downloading experience.

## Features

- **High-Quality Downloads**: Download videos in the highest available quality.
- **Playlist Support**: Download entire playlists with a single command.
- **Format Selection**: Choose from a variety of video and audio formats.
- **Easy to Use**: A user-friendly command-line interface makes downloading videos a breeze.

## Getting Started

To get started with YTDownloader, you'll need to have Python and `yt-dlp` installed on your system. You can install `yt-dlp` by following the instructions on their [official repository](https://github.com/yt-dlp/yt-dlp).

## Usage

To download a video, simply run the following command:

```bash
python main.py <video_url>
```

For more advanced options, such as downloading playlists or selecting specific formats, refer to the usage examples below.
