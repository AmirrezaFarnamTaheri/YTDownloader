# Contributing to YTDownloader

We welcome contributions to YTDownloader! Here are some ways you can contribute:

- **Reporting Bugs:** If you find a bug, please open an issue and provide as much detail as possible, including steps to reproduce the bug.
- **Suggesting Enhancements:** If you have an idea for a new feature or an improvement to an existing one, please open an issue to discuss it.
- **Pull Requests:** We welcome pull requests. Please make sure your code follows the existing style and that all tests pass.

## Development Setup

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/ytdownloader.git
    cd ytdownloader
    ```

2.  **Create a Virtual Environment**:
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows, use `venv\Scripts\activate`
    ```

3.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

4.  **Run the Application**:
    ```bash
    python main.py
    ```
